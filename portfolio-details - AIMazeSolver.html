About this project

This agent action based artificial intelligence was a learning experience in AI, solution search algorithms, and memory management. This program generates a maze full of gems and slimes for a little wizard to explore and hopefully escape. The little wizard is a representation of an agent, a program designed to take in data and make decisions, a simple form of artificial intelligence. This agent in particular has a list of actions it can take in this maze, namely picking up an item, moving through the maze, or attacking in any direction. The agent is omniscient (aka, aware of the entire maze) and chooses the best action to suit its priorities. In these examples it prioritizes making its way to the exit as efficiently as possible, picking up gems and slaying slimes along the way, though the code can be customized to prioritize seeking out every gem, slaying every monster, or maximizing points before exiting. I also learned in this project about fog of war, in this case the agent only being aware of tiles directly around it or places it’s already seen, but I haven’t yet created a smart enough decision tree for an agent to have that limited knowledge and escape consistently.
